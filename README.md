# 七夕トライアングル (Tanabata Triangle)

3人用の隠れんぼ系ボードゲームのブラウザ実装（ローカル1画面・パス回し）。

- **織姫・彦星（シーカー）**: 出会えれば勝ち。互いの盤面は見えず、軌跡の交差ヒントを頼りに近づく。
- **王様**: 二人を出会わせなければ勝ち。両盤面を見て、各シーカーの移動前にデブリを置いて妨害する。

## ルール

- 共有座標グリッド（9×9）上に織姫・彦星の2駒がいる。各シーカーは**自分の盤面だけ**を見る。王様は両方を見る。
- シーカーは1手で**上下左右にちょうど3マス**移動する（途中停止不可、自マスの重複・折り返し可、盤外・デブリへは進入不可）。
- 通ったマスには**軌跡**が残る。`織姫の軌跡 ∩ 彦星の軌跡` が**交差ヒント（★）**として各手番の開始時に開示される。
- 王様は各シーカーの移動前に、その盤面の**軌跡が無いマス**にデブリを1個置ける（2盤面で別々の位置に置ける）。
- 手番の流れ: `織姫デブリ → 織姫移動 → 彦星デブリ → 彦星移動 →（ヒント更新）→ …` を **7ラウンド**。
- あるシーカーが移動を終えた時点で**両者が同じマスに止まっていればシーカーの勝ち**。会えなければ王様の勝ち。

パス回しのため、各手番の切り替え時にハンドオフ画面を挟んで隠し情報の覗き見を防ぐ。

## 遊び方

ビルド不要。静的配信して `index.html` を開くだけ。

```sh
python3 -m http.server 8000
# → http://localhost:8000/index.html
```

（`file://` 直開きは ES モジュールの CORS 制約で動かないため、簡易サーバー経由で開くこと。）

## 構成

- `index.html` / `css/style.css` — 画面
- `js/engine.js` — ルールエンジン（純粋・DOM非依存の ES モジュール）
- `js/render.js` — Canvas 描画
- `js/app.js` — フェーズ進行・入力・ハンドオフ・勝敗
- `tests/engine.test.mjs` — エンジン単体テスト

### 調整可能な定数

`js/engine.js` の `DEFAULTS`（盤サイズ・移動マス数・ラウンド数・デブリ数・開始位置）で変更できる。

## テスト

```sh
node tests/engine.test.mjs
```
